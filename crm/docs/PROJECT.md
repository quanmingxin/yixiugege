## 功能模块

| 模块 | 描述 |
|------|------|
| 客户管理 | 客户信息录入、编辑、查询、分类 |
| 销售跟进 | 跟进记录、提醒、销售漏斗 |
| 数据分析 | 销售报表、客户分析、业绩统计 |
| 团队协作 | 权限管理、任务分配、团队沟通 |
